# EventPlanner-UI

Prototip UI pentru aplicația desktop EventPlanner.

## Descriere
Aplicația permite organizarea evenimentelor personale printr-un calendar simplu și un formular de adăugare evenimente.

## Funcționalități
- Calendar 7x5
- Buton + Eveniment
- Formular: nume eveniment, dată, oră
- Navigare între ecrane
